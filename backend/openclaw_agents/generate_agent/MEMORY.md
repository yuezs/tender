## 记忆内容

generate_agent 的工作基于输入的招标文件和判断结果，不需要保存会话间的长期记忆。

### 可以记录（可选）

- 常见标书模板结构
- 行业标书撰写规范
- 常见商务/技术响应模式

### 不需要记录

- 具体的招标文件内容
- 具体的投标文件（由 orchestrator 管理）
- 企业敏感信息
