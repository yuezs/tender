# 领导演示资料包

## 目录
- `knowledge/`：企业知识库示例资料
- `tender/`：可上传的示例招标文件
- `proposal/`：对应该项目的参考标书演示稿

## 推荐演示顺序
1. 在资料中心依次上传 `knowledge/` 下 5 份知识库文档
2. 对 5 份文档执行处理
3. 在招标处理页上传 `tender/tender_demo.docx`
4. 执行解析、抽取、判断、生成主链路
5. 在结果页展示提取字段、投标建议、目录和章节生成
6. 对照 `proposal/proposal_demo.docx` 讲解预期输出风格

## 说明
- 这套资料主题统一为“智慧园区综合管理平台建设项目”，便于整链路演示
- `tender/` 目录同时附带一个文本型 PDF 示例，可用于演示文本 PDF 解析能力
